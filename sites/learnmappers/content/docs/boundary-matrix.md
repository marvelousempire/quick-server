# Boundary Matrix (Advice vs. Education vs. Logistics)

**Version:** 1.0  
**Last Updated:** 2025-01-23

> Side-by-side table of prohibited requests and our educational substitutes.

## Purpose

Keep every task within the educational lane. This matrix clearly defines what LearnMappers does and does not provide.

## The Matrix

| Prohibited Request | Educational Alternative | Logistics Support |
|-------------------|------------------------|-------------------|
| Legal advice | Legal literacy education | Document organization |
| Tax advice | Tax system education | Record keeping |
| Financial advice | Financial system education | Data organization |
| Drafting documents | Document structure education | Template provision |
| Filing on behalf | Filing process education | Checklist creation |
| Representation | Process mapping | Meeting preparation |

## Key Boundaries

### ❌ We Do NOT:
- Provide legal, tax, or financial advice
- Draft, file, or represent
- Make recommendations
- Act as counsel

### ✅ We DO:
- Provide educational research
- Map systems and processes
- Create study materials
- Document findings

## Usage

Use this matrix to:
- **Kickoff** new projects
- **Review** Statements of Work (SOW)
- **Validate** scope boundaries
- **Maintain** compliance
